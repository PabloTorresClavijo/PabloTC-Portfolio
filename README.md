### 🔒 Seguridad (Headers & Hardening)

| Cabecera | Función |
|:---|:---|
| `Strict-Transport-Security` | Fuerza HTTPS permanente (HSTS) |
| `X-Frame-Options` | Previene ataques de Clickjacking |
| `X-Content-Type-Options` | Desactiva el sniffing de tipos MIME |
| `Content-Security-Policy` | Control estricto de recursos y scripts |
| `Permissions-Policy` | Restricción de acceso a hardware sensible |

<br>

🔐 **Medidas adicionales:** Gestión de variables de entorno segura, bloqueo de archivos sensibles y logs de errores monitorizados.
